# Simple RAG Backend - Document Q&A System

## A beginner-friendly Retrieval-Augmented Generation (RAG) system that allows you to upload documents and ask questions about them. Built with FastAPI, this project teaches the fundamentals of modern AI-powered document search and question answering.

## Project Structure

my-rag-project/
│
├── main.py # Main API server with all endpoints
├── test_api.py # Automated test suite
├── requirements.txt # Python dependencies
├── README.md # This file
└── test_doc.txt # Auto-generated test document

## Installation

### Prerequisites

### Python 3.8 or higher installed on your system

### pip (Python package manager)

### Install Dependencies

### pip install -r requirements.txt

## Running the Application

### python main.py

## API Documentation

### Once the server is running, open your browser and go to:

### http://localhost:8000/docs
